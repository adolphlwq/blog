# CS顶尖大学是如何培养计算机专业学生的
21世纪是科技革命的时代，计算机无疑是科技革命的核心引擎。计算机相关岗位的酬薪高于其它行业，因此，很多人通过培训或者自学等形式进入IT行业。

网上的资源和培训机构良莠不齐，让人眼花缭乱，本系列文章通过分析顶尖大学计算机专业培养学生的方案，总结出它们培养学生的范式和各自的特点。从顶尖大学的角度，为想转行CS的人提供一个方案。
